1. What is an array? A short and concise answer is expected here!

    2. An array is an ordered collection of elements that can be accessed by their index. It allows storing multiple values in a single variable.

  2. What does it mean to "hardcode" when we talk about programming? (In English, it's called "hard coding", "hard-coding", or "hardcoding").

    3. Hardcoding means embedding fixed values directly into the program instead of using variables, inputs, or data from external sources.

    5. Storing instances in arrays allows group operations, such as iterating, sorting, or filtering all objects efficiently. It also makes management and access of multiple similar objects easy.

  6. In task 8, we will use a "callback" routine when sorting. Explain how such a "callback" routine works.

    7. A callback in sorting is a function passed to the sort method. It receives two elements at a time and returns a number indicating their order: negative if the first should come before the second, positive if after, and zero if equal.
